Friday Night Bullet Arena Product Overview

Summary Sentence

Fast-paced top down round based local multiplayer shooting game with bouncing bullets.

Genre

Party, Local Multiplayer

Number of Players

2 to 4

Target Time for Average Play Session

Each round only lasts a few seconds. A game session might last up to 30 minutes.

Core Game Loop

  • Start the round.
  • Battle other players and win. (Only takes a few seconds.)
  • Congratulate the winner.
  • Begin the next round within 5 seconds.

Monetization

  • Initial purchase.
  • DLC maps.
  • DLC cosmetic changes for players/bullets.

Target Player

People with real-world gamer friends. (eg. Students, young adult gamers.)

Target Channels

itch.io

Go Live Date

Already live.

Platform

PC
